# Gaming Website (Frontend Practice)

A responsive multi-page gaming-themed website built with **HTML, CSS, and vanilla JavaScript**.

This project was created as a frontend practice project to demonstrate layout structure, styling, and basic interactivity.

## Live Demo
https://lynxgva.github.io/gaming-website/

## Pages
- Home
- About Us
- Contacts

## Features
- Responsive layout
- Custom CSS styling
- Basic form and email validation with JavaScript
- Multi-page navigation
- Clean HTML structure

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Purpose
This project is part of my frontend portfolio and was built to practice:
- Page layouts
- Flexbox
- Styling
- Simple DOM interactions
